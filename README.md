# Installation
Clone the github repository and update dependency
```sh
$ git clone https://github.com/chirag-prajapati-2705/laravel-app.git
$ cd laravel-app
$ composer install
$ npm install
$ npm run dev
```

   [Laravel Nova]: <https://nova.laravel.com/>
